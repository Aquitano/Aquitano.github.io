How to use

  1. CSS

    CSS-Tag: <link href="https://aquitano.github.io/Other/JustHTML/JustHTML.css" rel="stylesheet" />
    Minified CSS-Tag: <link href="https://aquitano.github.io/Other/JustHTML/JustHTML.min.css" rel="stylesheet" />

  2. JS

    JS-Tag: <script src="https://aquitano.github.io/Other/JustHTML/JustHTML.js"</script>
    Minified JS-Tag: <script src="https://aquitano.github.io/Other/JustHTML/JustHTML.min.js"</script>
