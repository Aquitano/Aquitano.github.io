As part of a geography seminar at the Adalbert Stifter Gymnasium, I developed the web platform **FrontiersToSpace**. It visualizes measurement data from a stratospheric balloon mission and makes the results accessible to a wider audience.

I was responsible for the entire **frontend** and **website architecture**. The stack includes **React.js** for interactive interfaces, **TailwindCSS** for styling and **Astro.js** for optimized load times. **Leaflet** provides interactive maps and **Chart.js** renders the collected sensor data as charts.

Beyond web development, I contributed to **building and programming the probe** -- specifically sensor integration and data transmission via APRS technology on the TTGO T-Beam board.

The project was a collaboration with the **German Amateur Radio Club** (DARC) and the Passau Observatory. It culminated on July 8, 2023 with a "Science Day" event hosted jointly at the school.

This project significantly improved my skills in **web development**, **teamwork** and **technical problem-solving**.
