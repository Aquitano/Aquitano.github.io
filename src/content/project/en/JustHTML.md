**JustHTML** is a **lightweight framework** that simplifies building simple websites and significantly reduces the need for separate JavaScript and CSS. It provides ready-made **CSS classes** and **JavaScript functions** that can be used directly in HTML.

The source code and latest release are available on the GitHub page.
